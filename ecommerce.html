E-commerce User Behavior Analysis

Predicting purchase intent and building recommendation systems using clustering algorithms

Role: Data Scientist
Timeline: 2 months (2023)
Tech Stack:
Python scikit-learn Flask t-SNE PCA

Project Overview

This project analyzes e-commerce user behavior patterns to predict purchase intent and build a recommendation system. Using a dataset of 50,000 user sessions from an online retailer, I applied clustering techniques to identify distinct user segments and their shopping behaviors.

Dataset and Business Value

The dataset included:

  • User browsing history (pages visited, time spent)
  • Demographic information
  • Purchase history
  • Cart abandonment patterns

This analysis helps businesses:

  • Identify high-value customer segments
  • Personalize marketing campaigns
  • Optimize product recommendations
  • Reduce cart abandonment rates

Technical Approach

I explored multiple techniques for clustering and visualizing user behavior:

k-Means Clustering

Partitioned users into distinct groups based on browsing and purchase patterns using the elbow method to determine optimal clusters.

DBSCAN

Identified dense clusters of similar users while detecting outliers who didn't fit standard patterns.

Dimensionality Reduction

Used t-SNE and PCA to visualize high-dimensional data in 2D/3D space while preserving relationships.

Key Findings

  • Identified 5 distinct user segments with different conversion rates
  • Discovered that "window shoppers" accounted for 60% of traffic but only 5% of purchases
  • Developed a recommendation system that increased conversions by 22% for target segments
  • Created a purchase intent prediction model with 85% accuracy

Implementation

The final system included:

  1. Real-time user segmentation as customers browse
  2. Personalized product recommendations via a Flask API
  3. Dashboard for marketing teams to monitor segments
  4. Automated alerts for high-intent users

Challenges & Solutions

Challenge: High-dimensional sparse data made clustering difficult
Solution: Used feature importance to select the most discriminative variables

Challenge: Imbalanced classes in purchase behavior
Solution: Applied SMOTE oversampling for the prediction model
